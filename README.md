# EjectCloud

Облачное файловое хранилище с поддержкой Telegram бота.

## Функции

- Загрузка и управление файлами
- Аутентификация через токены
- Административная панель
- Интеграция с Telegram ботом
- Квоты пользователей

## Запуск

```bash
mvn spring-boot:run
```

Приложение будет доступно по адресу: http://localhost:8080

## API

- `POST /api/auth/login` - Генерация токена входа
- `POST /api/files/upload` - Загрузка файла
- `GET /api/files` - Список файлов
- `DELETE /api/files/{id}` - Удаление файла
- `POST /api/admin/users` - Создание пользователя (админ)

## Конфигурация

Настройте `application.properties` для подключения к базе данных и Telegram боту.
